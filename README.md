# 🚀 AI-Powered Adaptive Learning Chatbot

## 📌 Overview

This project is an intelligent **AI-driven educational chatbot** built using **Flask, NLP (NLTK), and Machine Learning (Scikit-Learn)**. It is designed to provide structured, interactive, and context-aware learning experiences across multiple programming domains such as Python, Java, and C.

The system combines **rule-based logic, semantic similarity matching, and a Naive Bayes classifier** to deliver accurate and meaningful responses to user queries.

---

## 🧠 Key Features

* 🔐 **Secure Authentication System**

  * Manual login (Flask sessions)
  * Google OAuth integration

* 🤖 **Smart NLP Engine**

  * Tokenization, stopword removal, and lemmatization using NLTK
  * Custom preprocessing pipeline for better intent understanding

* 📊 **Machine Learning Integration**

  * TF-IDF vectorization
  * Multinomial Naive Bayes classifier for intent prediction

* 🔎 **Semantic Search Capability**

  * Character n-gram similarity for typo tolerance
  * Intelligent topic detection from structured JSON dataset

* 📚 **Structured Learning System**

  * Categorized topics (beginner → advanced)
  * Concept explanations with formatted outputs
  * Code examples for practical understanding

* 🧩 **Multi-Query Processing**

  * Handles multiple concepts in a single query
  * Combines results into a unified response

* 💬 **Chat History Management**

  * Save, retrieve, and delete chat sessions
  * Session-based storage

---

## ⚙️ Tech Stack

* **Backend:** Flask (Python)
* **NLP:** NLTK
* **Machine Learning:** Scikit-Learn
* **Authentication:** Authlib (Google OAuth)
* **Data Storage:** JSON-based knowledge system + session storage
